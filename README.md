# 网易云音乐云盘上传

解决 Mac 端网易云音乐无法上传到音乐云盘问题

## 下载安装

前往 [发布](https://github.com/ovnrain/netease-music-cloud-uploader/releases) 页面，下载 `.dmg` (x64: Intel 芯片，aarch64: Apple 芯片) 文件安装即可
